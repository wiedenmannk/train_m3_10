# Übung Tag 1 – HTML, CSS und Formulare

## Ziel

Baue eine kleine, saubere HTML-Seite mit externer CSS-Datei und einem einfachen Formular.

---

## Hauptaufgabe

Erstelle eine neue Seite mit dem Namen:

profil.html

Die Seite soll enthalten:

- eine Hauptüberschrift
- einen kurzen Begrüßungstext
- eine Liste mit mindestens 3 Punkten
- ein Formular mit folgenden Feldern:
  - Name
  - Alter
  - Lieblingsfarbe

---

## Anforderungen

- Binde eine externe CSS-Datei ein
- Die Überschrift soll blau sein
- Die Liste soll einen hellen Hintergrund haben
- Die Formularfelder sollen mit etwas Abstand untereinander dargestellt werden

---

## Mini-Aufgabe 1

Ändere die Navigation:

- füge einen Link auf `profil.html` hinzu

---

## Mini-Aufgabe 2

Ergänze im Formular ein weiteres Feld:

- Hobby

---

## Schlussaufgabe / Hausaufgabe

Erweitere `profil.html` so, dass die Seite etwas persönlicher wirkt:

- ergänze eine zweite Überschrift
- füge einen zweiten Absatz ein
- erweitere die Liste auf 4 Einträge
- gib dem Formular einen klaren Titel
- style den Button so, dass er besser sichtbar ist

---

## Hinweis

Achte darauf, dass alle Formularfelder ein `name`-Attribut besitzen.
