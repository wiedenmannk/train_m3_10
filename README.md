# M3.10 – Webentwicklung mit PHP

## Ziel

Dieses Projekt dient als Referenz für den Kurs M3.10.

Wir bauen Schritt für Schritt eine einfache Webanwendung:

Formular → PHP → Datenbank → Anzeige

---

## Struktur

01-html-css  
Grundlagen HTML, CSS und Formulare

02-php-basics  
Erste PHP-Skripte und Verarbeitung von Formulardaten

03-db-pdo  
Datenbankanbindung mit PDO und Speichern von Daten

04-crud  
Lesen, Bearbeiten und Löschen von Daten (CRUD)

exercises  
Übungen zu den einzelnen Tagen

---

## Nutzung

Projektordner:

C:\xampp\htdocs\m3-10

Start im Browser:

http://localhost/m3-10
