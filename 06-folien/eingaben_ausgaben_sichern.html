Eingaben & Sicherheit

Prüfen, absichern und sauber weiterverarbeiten

🔎 Eingaben prüfen

  • filter_input() validiert und bereinigt
  • z. B. Integer, E-Mail, URL
  • reduziert fehlerhafte Werte

🛡️ Ausgabe absichern

  • Problem: Code im Browser
  • <script>...</script>
  • Lösung: htmlspecialchars()

🗄️ Datenbank schützen

  • Problem: Manipulation von SQL
  • ' OR 1=1 --
  • Lösung: Prepared Statements
📌 Zusammenfassung:
Eingaben prüfen → filter_input()
Ausgabe absichern → htmlspecialchars()
Datenbank schützen → Prepared Statements
